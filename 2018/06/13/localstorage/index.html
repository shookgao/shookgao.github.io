localStorage 学习笔记

2018-06-13
localStorage, 学习笔记


localStorage 是 Web Storage 的一部分，用于持久化的本地存储，除非主动删除数据，否则数据是永远不会过期的；Web Storage 的另一部分是 sessionStorage，语法都一样，区别在于 sessionStorage 退出会话后数据就过期了。
Web Storage 并不安全，但可以用来存储一些不太敏感的数据。

目录

  • 增加
  • 删除
  • 修改
  • 查询
  • 复杂数据处理

增加

  • 语法：localStorage.setItem(key, value)
  • 示例：localStorage.setItem("name", "value")

删除

  • 删除单个值
    • 语法：localStorage.removeItem(key)
    • 示例：localStorage.removeItem("name")
  • 删除所有值
    • 语法：localStorage.clear()

修改

  • 和增加的语法是一样的，只要是同样的 key 就会覆盖。

查询

  • 语法：localStorage.getItem(key)
  • 示例：localStorage.getItem("name")
